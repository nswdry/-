## わかったこと💡
- Git管理下のファイルは削除したことも明示が必要！！！
  - `git add`ではインデックスに登録できない

```ruby
$ git rm (ファイル名)
```
この後`git commit`すると削除の登録ができる。`git add -u`でも可。
